# LDR-Logic-Based-Color-Detection-and-Sorting-System
LDR Logic-Based Color Detection and Sorting System. Microcontroller-free automation using Analog &amp; Digital Electronics. 8-color detection (RGB, CMYK, White/Black) sorted by a servo motor controlled by Op-Amps, a Decoder, and 555 Timers (PWM). Low-cost, purely hardware solution.
